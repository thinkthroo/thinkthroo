In this guide, you will learn the tooling used in [Lobechat](https://github.com/lobehub/lobe-chat).

# What's tooling?
You might be wondering what tooling means. If you check out [Bulletproof React's Project Standards](https://github.com/alan2207/bulletproof-react/blob/master/docs/project-standards.md),
it describes tools such as ESLint, Prettier, Husky etc.,

Similarly, in this guide, we provide an in-depth Lobechat tooling analysis.

# Lobechat tooling:

Lobechat is not a monorepo, you will find more information about Lobechat's project structure in project 
structure guide. We provide analysis of the tooling used:
1. [At root level](https://github.com/lobehub/lobe-chat) as there is only one Next.js project in Lobechat's repository,
you do not find  any additional workspaces or packages like in a monorepo. 

Tooling might vary based on an app/workspace or packages or a single Next.js project.
